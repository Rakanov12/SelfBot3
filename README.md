## INSTALL VIA GIT

```bash
> git clone https://github.com/takimtod/SelfBot23
> cd SelfBot3
> npm start
```

## INSTALL VIA TERMUX :

```bash
> pkg upgrade && pkg update
> pkg install nodejs
> pkg install bash
> pkg install git
> pkg install imagemagick
> pkg install ffmpeg
> pkg install libwebp
> termux-setup-storage
> cd /sdcard
> cd SelfBot3
> npm start
```
